# PICSUM
Photo gallery application with backend and frontend components

## Backend

### Dependencies
- Ruby
- Sinatra

### Installation
1. run `cd backend`
2. run `bundle install`
3. run `rackup config.ru`
4. navigate to `http://localhost:9292/api/v1/images`

## Frontend
### Dependencies
- Nodejs
- Reactjs

### Installation
1. run `cd web`
2. run `npm install`
3. run `npm start`
